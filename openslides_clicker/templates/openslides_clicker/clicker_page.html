Presentation Remote Interface

Simply by having this page opened, you wil be able to control the presentation with a presentation remote, a mouse or a keyboard. To ensure you will capture all click events when using a mouse, the standard OpenSlides UI has been stripped. To quit this page, simply click the 'back' button in your browser. It is recommended to use this page full-screen (F11).

Supported controls are:

Arrow LeftPrevious Page
Arrow RightNext Page
Left Mouse ButtonNext Page
